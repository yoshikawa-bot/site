Sobre a Yoshikawa

A Yoshikawa é um bot multifuncional projetado para tornar sua experiência no WhatsApp mais divertida e prática. Com mais de mil funções, ela pode:

  • Criar figurinhas a partir de imagens e vídeos
  • Baixar músicas diretamente no WhatsApp
  • Streaming de filmes, séries e animes
  • E muito mais, com comandos personalizados!

O uso no privado é gratuito, mas alguns comandos exclusivos estão disponíveis para usuários VIP. Use o comando !alugar para mais detalhes.

Tecnologias

  • Linguagens: Javascript + Node.js (v20)
  • Formato: CommonJS
  • Auxiliares: HTML + CSS
  • Conexão: Baileys-MD
  • Host: Shelter (Pterodactyl)
  • API: Proprietária
  • Criador: @kawalyansky
  • Versão: V5 (13/07/2025)
